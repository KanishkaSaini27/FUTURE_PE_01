---

## Prompt Logic

Each section of the website was generated using a **structured prompt** built around 5 key components:

| Component | Purpose |
|---|---|
| Role Assignment | Tell the AI to act as an expert copywriter |
| Business Context | Provide name, type, location, and audience |
| Tone Specification | Friendly and warm for a salon |
| Output Format | Specify exactly what to generate |
| Constraint | Keep it simple, persuasive, and website-ready |

## How to Reuse This Prompt System

This system is reusable for **any local business**. To adapt it:

1. Change **Business Name, Type, and Location**
2. Change **Target Audience**
3. Change **Tone** (friendly / professional / bold)
4. Update the **Services list** in the prompt
5. Run the same prompts — outputs adapt automatically
