# Prompts — Glowyy Salon Website Copy
**Task 1 | Future Interns Prompt Engineering Track**

---

## Prompt 1 — Homepage Copy

```
Act as an expert website copywriter specializing in local businesses.

Business: Glowyy Salon
Type: Full-service salon & beauty studio
Location: New Delhi, India
Target audience: Women aged 18–45
Tone: Friendly, warm, and confident — not corporate

Generate homepage website copy including:
1. A headline (clear value proposition, max 10 words)
2. A sub-headline (who it's for + key benefit, 1–2 sentences)
3. A short intro paragraph (3–4 sentences, persuasive, human-sounding)

Do NOT use generic phrases like "world-class" or "cutting-edge".
Make it feel real and relatable.
```

### Why I wrote this prompt this way:
I used role assignment first because it gives the AI a clear context to work from.
I added "Do NOT use generic phrases" because my first attempt was too vague and sounded fake.
I kept the tone instruction specific — "not corporate" — so the output felt like a real local salon.

## Prompt 2 — Services Page

```
Act as an expert website copywriter for local salons.

Business: Glowyy Salon (New Delhi)
Tone: Friendly, benefit-driven, simple

Generate a Services Page section with 4 services:
1. Hair Care (cuts, colour, treatments)
2. Skin & Facial Care
3. Bridal Makeup
4. Nail Art & Extensions

For each service write:
- A short service title
- A 2-sentence description (what's included + key benefit)
- One line: "Perfect for: [type of customer]"

Keep language simple and conversion-focused. Avoid jargon.
```

### Why I wrote this prompt this way:
I listed all 4 services clearly so the AI didn't invent random ones.
I added the "Perfect for" line to make each service feel targeted — I noticed without it the output was too generic.

## Prompt 3 — CTA Sections

```
Act as a conversion copywriter for a local beauty salon called Glowyy Salon in New Delhi.

Write 3 separate CTA (Call-to-Action) sections for the website:

1. Contact CTA — Encourage visitors to book a free consultation
2. Booking CTA — Drive immediate appointment booking (add urgency)
3. Trust-Building CTA — Build credibility with a social proof statement + CTA

Tone: Warm, confident, and friendly.
Each CTA should have:
- A short headline
- 1–2 sentences of supporting copy
- A button label in [ brackets ]
```

### Why I wrote this prompt this way:
I separated the 3 CTAs clearly so I got distinct outputs for each use case.
Specifying the button label format in [ brackets ] made the output directly usable on a real website.

## Prompt 4 — Tone Adaptation

```
Take the homepage headline and intro paragraph written for Glowyy Salon
and rewrite it in two alternate tones:

1. Professional / clinical tone (suitable for a medical clinic)
2. Bold and energetic tone (suitable for a Gen-Z fashion brand)

This shows how the same prompt structure adapts across different business types.
```

### Why I wrote this prompt this way:
I wanted to test how changing just the tone instruction completely changes the output.
This also shows the prompt system is reusable for any business type — not just salons.
